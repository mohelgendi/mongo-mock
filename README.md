MongoDB Service Mock


the API endpoints are:-

GET:/api/record/internalService?from=2010-05-05&to=2023-12-12

GET:/api/record (gets all records)

POST:/api/record (post new random record)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development w/ Nodemon
$ npm run start


# test w/ Jest
$ npm run test

```
